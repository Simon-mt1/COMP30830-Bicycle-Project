Auth Controller

auth_controller.py

Handles authentication-related routes including signup, login, and logout.

Routes:

  • /signup: Handles user registration.

  • /login: Handles user login.

  • /logout: Logs the user out of the session.

app.controllers.auth_controller.login()

Route to handle user login.

GET: Renders the login page. POST: Authenticates user credentials using AuthService. If valid, stores user in session and redirects to data dashboard; otherwise, reloads login page with error state.

Returns:

Rendered HTML template or redirect to data dashboard.

Return type:

Response

app.controllers.auth_controller.logout()

Route to log out the current user.

Pops the user from the session and redirects to the login page.

Returns:

Redirect to login route.

Return type:

Response

app.controllers.auth_controller.signup()

Route to handle user signup.

GET: Renders the signup page. POST: Processes signup form data using AuthService. If registration is successful, redirects to the login page; otherwise, reloads signup page with error state.

Returns:

Rendered HTML template or redirect to login route.

Return type:

Response
