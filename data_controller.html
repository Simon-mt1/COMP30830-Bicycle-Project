Data controller

data_controller.py

Defines routes related to maps, FAQs, and the homepage.

Routes:

  • /map: Displays the Google Map with bike stations and weather info.

  • /index: Dashboard/homepage after user login.

  • /: Redirects to the login page.

  • /faq: Displays the FAQ page.

app.controllers.data_controller.faq()

Route to render the FAQ page.

Returns:

Rendered FAQ HTML page.

Return type:

Response

app.controllers.data_controller.index()

Route to render the main dashboard page.

Only accessible if the user is logged in.

Returns:

Rendered index page or redirect to root.

Return type:

Response

app.controllers.data_controller.map()

Route to display the interactive map with bike station and weather data.

Checks if the user is authenticated, fetches JCDecaux bike data and weather data, formats them, and renders the map page. Redirects to login if user not in session.

Returns:

Rendered map HTML page or redirect to login page.

Return type:

Response
